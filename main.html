ยินดีต้อนรับเข้าสู่เว็บไซต์เพื่อการศึกษาสำหรับการเรียนการสอน วิชาพื้นฐานการสร้างเว็บไซต์


คำอธิบายรายวิชา ศึกษาและปฏิบัติเกี่ยวกับโครงสร้าวเว็บไซต์ หลักการออกแบบเว็บไซต์ หลักการออกแบบเว็บไซต์และสร้างเว็บไซต์ด้วยโปรมแกรมภาษา HTML 5 การตกแต่งข้อความ การแทรกรูปภาพหรือสื่อต่างๆ การสร้างตาราง การสร้าง การเชื่อมโยงในแบบต่างๆ การสร้างเฟรม การออกแบบและสร้างฟอร์ม การแทรก Java Script การทดลองการทำงานของเว็บไซต์ การอัพโหลกเว็บไซต์ไปยังเว็บเซร์ฟเวอร์
